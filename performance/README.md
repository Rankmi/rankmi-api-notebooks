# Performance API
El API de Desempeño permite la consulta de todos los procesos, los colaboradores involucrados y los resultados obtenidos en cada proceso que las empresas inician, ejecutan y finalmente cierran dentro de Rankmi.

# Documentación
[Aqui la documentación completa del API de Desempeño](https://performance-reports-api.rankmi.com/api-doc)

# Métodos e Información disponibles
- Autenticación
- Colaboradores
- Procesos empresariales
- Resultados Globales
- Resultados Individuales
- Distribución Global de Resultados
- Jerarquía de áreas

# Tecnología
GraphQL

# Lenguaje (de los ejemplos en este repositorio)
Python 3.0

# Necesidades iniciales
Para el uso del API y la ejecución de los ejemplos descritos en este repositorio, la empresa deberá contar con lo siguiente:
- Secret Token (este token será entregado por Rankmi a la empresa, o puede ser obtenido desde la configuración administrativa de la empresa)
- El ID de algún Proceso ó el Token de algun Proceso y/ó el Token de algún Usuario en particular
